# PRODIGY_CS_03: Password Complexity Checker

A web-based tool built during my Cyber Security Internship at ProDigy Infotech that assesses the strength of a password based on key security criteria.

## Features
- Real-time password evaluation using regular expressions.
- Interactive checklist covering length, uppercase, lowercase, numbers, and symbols.
- Dynamic, color-coded visual strength bar (Weak, Medium, Strong).
- Toggle visibility button (Show/Hide password).

## Tech Stack
- HTML5
- CSS3 (Dark-mode aesthetic)
- Vanilla JavaScript
